# STD_FANH

This repository contains notes, data products, and Python workflows for studying axion dynamics during a first-order phase transition.

## Layout

- `ode/`: homogeneous ODE solvers, scans, and analysis scripts.
- `lattice_data/`: geometry inputs and tabulated datasets used by the studies.
- `axion_pt.md`: project overview and physics notes.
- `axion_pt_semi_analytic.md`: semi-analytic derivation notes for the anharmonic factor fit.

## Notes

- Vendored Julia environments and Julia package depots were removed from the repository.
- The analysis tree still contains `PySR` outputs and the Python driver script that produced them.
